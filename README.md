## Simple Notes
### Идея 
Простое приложение для заметок со встроенной базой данных, в которой они хранятся.
### Функционал
- Создание, изменение, удаление заметок
### Стек
ЯП: Kotlin
IDE: Android Studio
Библиотеки: Retrofit2, MVVM, RecyclerView, Room
